# NemCSS

> A design-token-driven CSS utility generator

NemCSS is a utility that helps you generate CSS custom properties and utility classes based on your design tokens.

The idea is that you define your design tokens (for fonts, spacings, and so on) and with `NemCSS`, you get custom properties and a series of utility classes directly available.
You can configure which CSS utility classes get generated, the naming conventions of both custom properties and utility classes with one simple `nemcss.config.json`.
